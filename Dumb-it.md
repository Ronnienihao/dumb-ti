# DUMB-TI MVP 需求规格说明书 (全球版 v1.1)

## 1. 产品定位
*   **目标市场**：欧美 Gen Z & Alpha（主打 X, INS, Reddit, Snapchat）。
*   **核心策略**：通过 Meme 文化驱动的“毒舌”心理测试，利用欧美社交媒体传播逻辑完成裂变。
*   **核心价值**：提供“又狠又准”的人格标签，提供社交货币。

## 2. 核心功能
*   **心理测试引擎**：24 道逻辑题，基于权重分计算 16 种人格归属。
*   **内容体系**：
    *   **测试题库**：24 题（用户自行上传/维护 JSON）。
    *   **人格插画**：16 张用户预设好的高质量插画（直接存放于项目 `/public/assets/personalities/`）。
*   **社交裂变系统**：结果页提供“一键分享”按钮，适配 X/INS/FB 尺寸，分享即解锁高级版解析。

## 3. 技术方案
*   **框架**：Next.js + Tailwind CSS + Framer Motion。
*   **部署**：Cloudflare Pages (支持多域名映射，方便 SEO 铺设)。
*   **数据结构**：采用 JSON 配置文件存储题目与人格映射，支持随时热更新。

## 4. 用户流程
1.  **触达**：多域名（十几个）SEO 导流。
2.  **首页**：Logo (Dumb-ti) + "Let's Gooooo!" 按钮。
3.  **测试**：24 道题循环展示，带进度条，每页一题。
4.  **结果**：展示 16 型人格描述，加载对应的静态插画，并在图片/文字层植入水印。
5.  **转发**：一键转发按钮，完成裂变，引导至 Pro 深度解析（5 天活动限制）。

## 5. 商业化策略
*   **标准版 (Standard)**：24 道题测试 + 基础人格分析 + 静态结果图（带版权/网站水印）。
*   **Pro 版 (Premium)**：48 题模型 + 深度建议（恋爱/职场/生活）。
*   **裂变白嫖策略**：用户将结果分享至社交媒体并截图/点击分享链接，系统发放 5 天 Pro 版使用权（MVP 阶段用引导逻辑实现）。不转发者需支付 $4.9 获取永久权限。

## 6. 开发与部署计划 (2天)
*   **Day 1**：搭建 Next.js 脚手架，实现题库 JSON 逻辑、前端界面渲染，完成静态插画加载系统。
*   **Day 2**：配置 Open Graph (OG) 标签以保证社交分享预览效果，连接 Cloudflare，进行上线测试。

## 7. 风险评估
*   **SEO 合规**：配置 `canonical` 标签，避免多域名产生重复内容权重分散。
*   **图片加载**：插画统一存放在公共目录，确保 CDN 缓存有效性。
*   **传播机制**：针对 X/INS 分享逻辑预留 OG Meta 接口，确保分享后的预览图就是你设计的插画图。
